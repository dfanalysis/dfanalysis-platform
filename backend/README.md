# Backend - Plataforma DF Analysis IA

Backend oficial da Plataforma DF Analysis IA.

## Stack

- FastAPI
- PostgreSQL
- SQLAlchemy
- Alembic
- Docker
- Redis
- OpenAI

## Executar

Em desenvolvimento:

```bash
docker compose up
```
