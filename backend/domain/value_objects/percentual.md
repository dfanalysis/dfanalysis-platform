# Percentual

## Objetivo

Representa um valor percentual genérico utilizado em regras de negócio.

## Diferença para Alíquota

A Alíquota possui significado tributário.

O Percentual representa qualquer percentual do domínio.

Exemplos:

* taxa administrativa;
* participação societária;
* comissão;
* rateio;
* desconto.

## Regras de Negócio

* Deve possuir valor decimal.
* Deve ser imutável.
* Pode assumir qualquer valor permitido pela regra de negócio.

## Observações

Nem todo Percentual representa imposto.

Por isso ele deve existir separado do conceito de Alíquota.
