# Alíquota

## Objetivo

Representa um percentual utilizado para cálculo tributário.

## Estrutura

Exemplos:

* 2,00%
* 5,00%
* 11,25%

## Regras de Negócio

* Deve possuir valor decimal.
* Não pode ser negativa.
* Deve ser imutável.

## Uso no Domínio

Pode representar:

* ISS;
* IBS;
* CBS;
* retenções;
* tributos federais;
* tributos estaduais;
* tributos municipais.

## Observações

A Alíquota representa apenas o percentual.

O cálculo do imposto pertence ao domínio fiscal.
