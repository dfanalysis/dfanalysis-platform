# OperationalCase Aggregate

## Responsabilidade

Representar o processo operacional.

Não representa os dados especializados.

Representa apenas o ciclo de vida.

---

# Deve conhecer

Tipo

Status

Etapa

Origem

Competência

Empresa

Comunicação

Interpretação

---

# Não deve conhecer

ISS

NFS-e

XML

Banco

Pagamento

Repasse

Procedimentos

Médicos

---

# Objetivo

Ser o ponto central de rastreabilidade da Plataforma DF Analysis IA.

Todo domínio especializado deverá relacionar-se ao OperationalCase.

Ele representa o processo.

Os demais domínios representam as especializações.
